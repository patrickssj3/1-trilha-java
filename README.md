# 🏦 Projeto ContaBanco

Este é um projeto prático em Java desenvolvido com o objetivo de exercitar os conceitos fundamentais da linguagem, abordando desde a sintaxe básica até a interação com o usuário via terminal.

## 📚 Tópicos Aprendidos e Praticados

* **Sintaxe Java:** Declaração correta de variáveis utilizando diferentes tipos de dados (`int`, `double`, `String`).
* **Classe `Scanner`:** Captura de dados inseridos pelo usuário via terminal, lidando com diferentes tipos de entrada (`nextInt`, `nextDouble`, `nextLine`) e dominando o comportamento de buffer de teclado (resolução da quebra de linha após leitura de números).
* **Manipulação de Strings:** Uso prático da concatenação para unir textos e variáveis, gerando uma mensagem final formatada e dinâmica.
* **Configuração de Ambiente:** Ajuste de versões do JDK (Java Development Kit) no Visual Studio Code, garantindo a compatibilidade correta (Java 17 / 21) para compilação e execução.

This is a practical Java project developed to exercise fundamental language concepts, ranging from basic syntax to user interaction via the terminal.

## 📚 Topics Learned and Practiced

* **Java Syntax:** Correct variable declaration using different data types (`int`, `double`, `String`).
* **`Scanner` Class:** Capturing user input through the terminal, handling various input types (`nextInt`, `nextDouble`, `nextLine`), and mastering keyboard buffer behavior (resolving the newline issue after reading numeric values).
* **String Manipulation:** Practical use of concatenation to join text and variables, generating a formatted and dynamic final message.
* **Environment Setup:** Adjusting JDK (Java Development Kit) versions in Visual Studio Code, ensuring proper compatibility (Java 17 / 21) for compilation and execution.
